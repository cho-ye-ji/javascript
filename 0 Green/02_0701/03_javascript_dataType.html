3 자바스크립트 데이터의 유형 종류

  • 문자데이터(string)
  • 숫자데이터(number)
  • 부울형 (boolean) - true(확인), false(취소)
  • 찾을 수 없는 데이터형(undefined)
  • 값이 없는 데이터(null)
  • 배열형 데이터(array)
  • 함수형 데이터(function)
  • 객체(object) - document, window, alert, confirm ...

typeof - typeof 연산자는 사용자가 입력한 데이터가 어떤 종류의 형식인지를 알려주는 연산자이며, 변수앞에 Number, String등을 작성하면 실제 데이터 형식을 변활할 수 있다.
