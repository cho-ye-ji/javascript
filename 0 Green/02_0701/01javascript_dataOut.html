자바스크립트에서 데이터 출력 방법

자바스크립트에서 다양한 방식으로 데이터를 출력할 수 있다.

  • 1. document 객체에 write 메소드를 사용하여 내용 출력하기 - document.write('출력할 문자 내용');
  • 2. window 객체를 사용하여 메세지 띄우기 - window.alert('메세지 띄우기')
  • 3. 브라우저 개발자 모드(f12) console 모드에서 내용 출력하기 - console.log(출력할 내용)/비공개적 테스트
  • 4. document 객체의 innerHTML 메서드로 내용 출력하기 - document.getElementById('id명').innerHTML ='출력할 내용';
