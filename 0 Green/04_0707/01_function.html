01_function(함수) p.131

실행할 내용을 재사용이 가능하게 묶어둔(모듈화) 프로그램 선언
입력(인자,argument)과 출력(반환, return)으로 구성되어 있음
사용자 정의 함수, 내장 함수가 있음
함수에 저장된 일련의 실행문은 함수 호출문에 의해 언제든 실행할 수 있습니다.

함수(function)란?
함수는 일련의 내용을 삽입하여, 사용자가 언제든 호출하여 내용을 실행할 수 있는 실행문이다.
언제든 사용자가 필요할 떄 다시 실행할 수 있다. 코드의 중복제거 및 재사용이 용이하다.
유지 보수의 용이성이 높다.
함수의 종류에는 내장 함수(prompt, alert....등),p154와 사용자 정의 함수가 있다. 
        1.매개변수가 없는 일반 함수
        function 함수명(){
            실행할 내용;
        }

        2.매개변수가 있는 일반 함수
        function 함수명(변수1,변수2,변수3){
            실행할 내용;
        }
        
        3.리턴값이 있는 함수
        function 함수명(변수1,변수2,변수3){
            실행할 내용;
            return 실행결과;
        } 

        es6 문법에서는 함수를 좀 더 일기 쉽고, 체계적이고, 간략하게 작성할 수 있다. (화살표 변수, function 대신 화살표 생김)

        let 변수명 = (변수) => {
            실행할 내용;
        }
