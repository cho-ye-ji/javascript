실습01_함수를 활용한 이미지 배너 슬라이드

이미지 슬라이드 박스 안에 좌,우 저튼 클릭시 페이지 수 푶시하기
구현순서
1. banner 박스 영역을 만들고 이미지 태그를 삽입하여 첫번째 이미지 삽입
2. 아이콘 폰트를 활용하여 좌,우 방향 버튼을 삽입
3. 자바스크립트에서 nextImg, prevImg 함수를 각각 작성하여 총 이미지의 갯수를 4장을 초과하지 않도록 조건문(if)을 작성한다.
4. 좌,우 방향 버튼을 클릭시 각각 함수명을 호출하여 해당 이미지가 나오도록 한다.
5. 오른쪽 하단에 페이지 번호를 표시하는 내용 작성
