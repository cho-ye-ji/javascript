01 javascript 기초 문법 p.15

  1. 자바스크립트는 html 문서 안에서 삽입되어 작동된다.
  2. 기본적으로 css 언어와 같이 head안에서 작성되며, 필요에 따라 body 태그 안에서도 작성이 가능하다.
  3. 주석을 사용할 때 2가지 종류가 있으며, 한줄 주석은 '//주석내용', 2줄 이상의 주석은 '/* 여러줄 주석내용 */' 기호로 사용된다.
  4. HTML,CSS 태그 요소는 대소문자를 구별 안하지만, 자바스크립트는 대,소문자를 구별한다.
  5. 자바스크립트는 문자 데이터를 입력시 '' 홑따옴표 기호, " "쌍 따옴표 기호를 사용하여 묶어준다.
  6. css처럼 문서내에서 작성하는 것을 1.내부 스크립트라고 하며 외부 문서로 별도 저장하여 html 문서에 삽인하여 연결하는것을 2. 외부스크립트(파일명.js)라고 한다.
  7. 자스의 선언은 script, /script 로 선언하고 종료한다.
